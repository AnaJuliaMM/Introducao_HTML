*Atributos das tags:
Os atributos são propriedades que as tags tem. Cada tag terá seu grupo de atributos compatível, mas existem três atributos que são padrão: "id", "style" e "class".Uma mesma tag pode ter um ou todos esses.

O id é um identificador de um elemento.Através dele podemos controlar a tag qeu tem esse nome via JavaScript, por exemplo. 

A class serve para padronização, pois todos os elementos que possuem uma mesma classe, receberão um mesmo grupo de característiccas, que podem ter sido definidas via CSS, por exemplo.

E o atributo style é a possibilitade de estilizar um elemento(cor, tamanho, posição, etc). Dessa forma, realiza-se a função do CSS direto no HTML. Contudo, a título de boa prática, normalmente, prefere-se realizar a estilização CSS num arquivo a parte.
